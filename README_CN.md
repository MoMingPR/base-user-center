# 用户中心基础模板
这是一个基础的用户中心管理网站模板.
一个简单的实现用户登录、注册、查询等基本功能的用户管理系统。
## 技术选型

## 前端：
主要运用到阿里 Ant Design 生态。
 + HTML + CSS + JavaScript
 + React 开发框架
 + Ant Design Pro 项目模板
 + Ant Design 端组件库
 + Umi 开发框架
 + Umi-Request 请求库

 ## 后端:

 + Java 编程语言
 + Spring + SpringMVC + SpringBoot 框架
 + MyBatis + MyBatis Plus 数据访问框架
 + MySQL 数据库
 + jUnit 单元测试库
